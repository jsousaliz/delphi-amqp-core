## Diretrizes

- Mantenha a API publica baseada em interfaces.
- Evite dependencias externas obrigatorias.
- Preserve compatibilidade com Delphi 10.4+ Win64.
- Adicione testes para novos comportamentos de protocolo.
- Documente mudancas publicas no [README](README.md) ou em [`docs/`](docs/README.md).

## Fluxo recomendado

1. Crie uma branch pequena e focada.
2. Implemente a mudanca com testes.
3. Valide o exemplo quando a mudanca afetar uso publico.
4. Abra um pull request descrevendo comportamento e impacto.
