# Etapa 8: Projeto de Exemplo

## Entregas

- Projeto separado em `examples/`.
- Conexao com RabbitMQ local.
- Criacao de fila, publish, consume, purge e delete.
- Codigo didatico.
- Separar no exemplo comandos/rotinas para cada fluxo: conectar, declarar fila,
  publicar, consumir, limpar e deletar.
- Exibir logs de forma legivel e deixar claro quais callbacks rodam em worker
  thread.
- Incluir configuracao visivel para host, porta, vhost, usuario e senha.

## Definicao de pronto

- Exemplo abre no Delphi 10.4+ Win64.
- README documenta execucao.
- Exemplo executa de ponta a ponta com RabbitMQ local usando as credenciais
  padrao `guest`/`guest`.
